### 剪切 复制
```
yy ：复制光标所在的这一行
4yy ：复制4行
p ：粘贴代码到光标下一行
dd ：剪切光标行
2dd ：剪切两行
D : 从当前光标到行末剪切
D0 : 从当前光标到行首剪切
x : 删除光标后一个字符
X : 删除光标前一个字符
```

### 定位
```
h 左 j 下 k 上  l 右 
H : 当前屏幕上方
M : 当前屏幕中间
L : 当前屏幕下方

Ctrl+f 向下翻一页代码
Ctrl+b 向下翻一页代码
Ctrl+d 向下翻半页代码
Ctrl+u 向上半页代码

20G ：快速定位到第二行代码
G ：快速的回到整个代码的最后一行
gg ：快速回到整个代码的开头

w : 向后跳一个单词的长度
b : 向前跳一个单词的长度

[[ ：到首行
]] ：到末行
{ : 到上一块代码
} : 到下一块代码
```

### 代码移动
```
选中一片代码：
v ：选到光标处
V : 选到光标行
>> :往右移
<< :往左移
. ：重复执行上一次命令 连续移动代码
```

### 搜索 替换
```
搜索
/hello + enter:
n :上一个查找到的词
N ：下一个查找到的词

替换
末行模式下当前文件中所有的abc替换成123
：%s/abc/123/g
末行模式下，将一行到第10行之间的abc替换成123
：1,10s/abc/123/g
```

### 保存退出
```
wq : 保存退出
Shift+zz :保存退出
```
